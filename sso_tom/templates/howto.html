How to use the SSO Alert System

Introduction

This page provides basic instructions on how to use the SSO Alert System.

This system provides the ability to automatically submit observations to the ANU 2.3m as well as other facilities.

By the end of this guide, you should be able to submit an observation to the ANU 2.3m

In order to use this service you should be familiar with the "Applying for time on the ANU 2.3-metre telescope" page.

This instance of the SSO Alert System is deployed by the Research School of Astronomy and Astrophysics at the Australian National University

For additional assistance or enquires please contact: sso-alert-system@mso.anu.edu.au
